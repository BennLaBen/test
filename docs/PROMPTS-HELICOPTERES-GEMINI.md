# Prompts pour générer les 20 images d'hélicoptères sur Gemini

## Instructions

1. Va sur **gemini.google.com** avec ton compte Pro
2. Copie-colle chaque prompt ci-dessous (un par un)
3. Télécharge l'image générée
4. Renomme-la avec le nom de fichier indiqué (ex: `h120.jpg`)
5. Place toutes les images dans : `public/images/aerotools/helicopters/`

---

**Fichier : `h120.jpg`**

```
Generate a photorealistic image of an Airbus H120 / Eurocopter EC120 Colibri helicopter. It is a light single-engine civilian helicopter with a sleek white and blue corporate livery, polished finish. The helicopter is parked on a concrete tarmac helipad. Directly behind it is a large modern aviation hangar painted in dark corporate blue color (like Airbus blue). Clear sunny day, professional aviation photography style, sharp focus, high detail, 16:9 aspect ratio. No text or logos.
```
