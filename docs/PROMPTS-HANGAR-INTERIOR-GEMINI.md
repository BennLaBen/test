# Prompts Gemini — Images Hangar Interior (Boutique)

## Instructions

1. Va sur **gemini.google.com** avec ton compte Pro
2. Copie-colle chaque prompt ci-dessous (un par un)
3. Télécharge l'image générée
4. Renomme-la avec le nom de fichier indiqué
5. Place toutes les images dans : `public/images/aerotools/hangar/`
